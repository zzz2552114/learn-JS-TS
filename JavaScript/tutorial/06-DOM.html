这次我们来学习DOM, document object model
实际上就是学习“Document”这个内置对象
用来操作网页的HTML

1.一般来说我们用document.querySelector("")就够了，这会给我们返回一个html元素的object对象
2.然后我们document.querySelector("").innerHTML或者document.querySelector("").innerText就可以获取、修改内容了
3.querySelector里面一般是写id或者class，写id的话是"#id", 写class的话是".class"
4.如果document的操作放在函数里那无所谓，否则要在操作对象加载后再写，就是放在操作对象的下面
5.在同一个代码块下可以用this操作。比如button的onclick下面可以用this指代document.querySelector("")
6.注意，document.querySelector("").innerHTML或者document.querySelector("").innerText格式是字符串，如果要修改，最后是渲染到html上的，换行是br
